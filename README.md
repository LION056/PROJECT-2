# Email Authentication Flow

A modern authentication experience built with HTML, CSS, and JavaScript.

## Live Demo

https://danielobute.github.io/BudgetBasket/

## Features

* Email validation
* OTP verification flow
* Existing user authentication
* New user onboarding
* Dashboard screen
* Responsive design
* Progress indicators
* Success and error states
* Resend verification code
* Auto-focus OTP inputs

## Technologies Used

* HTML5
* CSS3
* JavaScript (Vanilla JS)

## User Flow

### Existing User

Email → OTP Verification → Dashboard

### New User

Email → OTP Verification → Onboarding → Dashboard
